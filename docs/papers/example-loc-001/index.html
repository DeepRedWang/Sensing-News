UWB-SLAM: Tightly Coupled UWB-Inertial SLAM for GPS-Denied Environments

◎ 定位 IPSN 2026 2026-05-17 7.9

David Smith, Eve Johnson

UWBSLAMIndoor LocalizationIMU
arXiv Code
TL;DR

UWB+IMU 紧耦合因子图 SLAM，稀疏锚点下也鲁棒。

关键贡献

  • 稀疏 anchor 下仍可定位
  • 完全开源的工厂级测试集
  • 实时性 50 Hz 以上

解读

贡献： 在稀疏 anchor（≤3 个）情况下传统 trilateration 失效。作者引入 IMU 预积分约束，并通过滑动窗优化解决 anchor 不可观条件下的初始化问题。

实验： 在 800 m² 仓库环境下，3 个 anchor 时定位 RMSE 12 cm，相比 EKF baseline 提升 40%。

原始摘要

We propose UWB-SLAM, a tightly coupled factor-graph framework fusing UWB ranges with IMU pre-integration for robust localization in GPS-denied environments with sparse anchors.
